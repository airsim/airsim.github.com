How to use these images

You will need a virtual machine manager. If you are not familiar with these, we recommend Oracle Virtual Box.

After you downloaded the image you chose, you can easily import your image/appliance. All these images are preconfigured but feel free to allow more ressources in your VM manager for a smoother TvlSim experience.
