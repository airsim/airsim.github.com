Getting started

There are many ways to get started with AirSim and this page is here to give you all the information you need to do so easily.

AirSim comes in different flavors :
  • Online
  • Pre-installed in a downloadable virtual machine
  • Packaged

Choosing which flavor is the right for you is pretty easy. If you are looking to use AirSim as soon as possible in a preconfigured environment you can go to the #online version or download the #preconfigured virtual machine and import it into Virtual Box. If your goal is to have the most amount of fun, start contributing or simply dig deep into AirSim, you can get the source code on #Github.
