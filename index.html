AirSim is an open-source Airline Market Simulator built around independant modules. Enjoy it as a whole or use a few modules, it is powerful and flexible.

What is it ?

The Airline Market Simulator (AirSim) is an open source demand generation tool originally developed in C++ by the Amadeus IT Group. You are therefore free to use and modify it according to the GPL3 license terms.

What does it do ?

Simulate an airline market is a bit vague. It will generate flight, demand, populate the inventory, make coffee, mow the grass, save your soul, make you rich, help the poor and much more.

How can I contribute ?

AirSim is open source meaning you can download it and modify it freely. For more information you can go to the documentation page where guides and tutorials will help you in your first steps relating to configuration and usage. If you want to jump in right in, you can go to the download page and get the source files, or you can get a fully configured virtual machine and import it in your favorite VM manager.
