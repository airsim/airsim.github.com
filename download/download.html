GitHub


Want the whole Travel Simulator or one of the modules? Everything is available on the GitHub project page. If you are not sure what you need to clone the documentation is here for you.

Preconfigured Virtual Machine


Want to use Travel Simulator right away? Download a virtual machine preconfigured with all the modules Travel Simulator has to offer. Please note that although the virtual machine image is updated frequently make sure that you pull the latest version of Travel Simulator from GitHub by launching the script you will find on the desktop.

To use the virtual machine you will need to import it in VirtualBox. A step by step tutorial is available to help you in the guides.
