Only want to use AirSim?

So you want to use AirSim but you don't want to run it localy on your computer? Nothing easier, go to this page and use AirSim right away, in your browser. (A good one however)

GitHub

Want the whole AirSim or one of the modules? Everything is available on GitHub. If you are not sure what you need to clone the documentation is here for you.

GitHub project page.

Preconfigured Virtual Machine

Want to use AirSim right away? Download a virtual machine preconfigured with all the modules AirSim has to offer. Please note that although the virtual machine image is updated frequently make sure that you pull the latest version of AirSim from GitHub by launching the script you will find on the desktop.

To use the virtual machine you will need to import it in VirtualBox. A step by step tutorial is available to help you in the guides.
